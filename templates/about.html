Efficient Frontier

This website have the intent to show my skills with Finance and Programming languages to create, reproduce and deploy my own solutions besides that i'm not too concern about the front-end.

I made a Flask application to deploy a Markowitz model that calculate Efficient Frontier of 10.000 simulated portfolios, using libraries Numpy, Pandas, Plotly, Yahoofinance...

The efficient frontier from Markowitz explain that given a N number of stocks in a portfolio, how could the mannager know how much to buy of each stock to get the better return given some level of risk.
